# 📖 Le Petit Bon Appétit - Digital Recipe Book

This repository contains the custom **HTML and CSS** layout designed for the landing page or featured section of the **Bon Appétit** cooking blog.

The interface is inspired by a vintage cookbook aesthetic, featuring elegant typography from Google Fonts (`Playfair Display`, `Pinyon Script`) and a warm, nostalgic color palette.

---

## ✨ Design Features
* **Vintage/Parchment Style:** A main container with subtle, aesthetic borders that mimic the feel of a physical book page.
* **Responsive Layout:** A clean, adaptive grid (`CSS Grid`) that beautifully organizes recipe buttons across all screen sizes.
* **Smooth Hover Effects:** Elegant transitions and lift effects when interacting with the navigation elements.
* **Cozy Color Palette:** Cream base (`#F9F6F0`), dark brown typography (`#5C4033`), and soft strawberry-pink accents (`#E79A8F`).

## 🛠️ Built With
* HTML5
* CSS3 (Native Custom Properties / Variables)
* Google Fonts
